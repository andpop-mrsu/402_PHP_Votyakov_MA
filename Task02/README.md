## Описание

Веб-приложение игры "Арифметическая прогрессия" с сохранением истории игр в базе данных SQLite.

## Установка и запуск

1. Клонировать репозиторий:
```
git clone https://github.com/Mih-gif/402_PHP_Votyakov_MA.git
cd Task02
```

2. Запустить веб-сервер:
```
php -S localhost:3000 -t public
```

3. Открыть в браузере:
```
http://localhost:3000/
```

4. Функционал:
```
Угадывание пропущенного числа в арифметической прогрессии
Сохранение истории игр в базе данных
Просмотр истории всех игр
```

5. Используемые технологии:
```
PHP 7.4+

SQLite 3

HTML5, CSS3
```
